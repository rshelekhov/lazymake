This directory will contain GIF recordings and screenshots for the README.

See the HTML comments in the main [README.md](../../README.md) for detailed recording instructions for each GIF:

## Recording Tools

Recommended tools for creating terminal GIFs:

- **[VHS](https://github.com/charmbracelet/vhs)** - Declarative terminal recording (recommended)
- **[asciinema](https://asciinema.org/) + [agg](https://github.com/asciinema/agg)** - Record and convert to GIF
- **[terminalizer](https://terminalizer.com/)** - Another good option

## Specifications

- **Terminal size**: 100x30 (width x height)
- **File size**: Optimize to <5MB per GIF
- **Width**: 800px when displayed
- **Format**: Animated GIF
